How to apply the corrected file in FLEx

  1. Click Download modified file — saves with _ws_corrected in the filename.
  2. In FLEx, go to File → Import → FLEx Text Interlinear…
  3. Select the downloaded file.
  4. If FLEx asks whether to merge with existing texts, choose Merge (or Replace).
  5. Verify the changes in your texts before saving.

The tool replaces lang="old-code" with lang="new-code" everywhere within selected texts — phrase, word, morpheme, gloss, and translation lines, plus the <languages> metadata block. File structure, content, and version number are preserved exactly.
